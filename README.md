# IT-notebook-app

Ứng dụng sổ tay điện tử dành cho lập trình viên, hỗ trợ lưu trữ công thức, thuật toán, SQL, Regex và ..... Xây dựng bằng Java, kiến trúc MVVM, Room Database và Navigation Component.

## Hướng dẫn khởi chạy


Sau khi clone repo về, chạy các lệnh sau tại thư mục gốc của project:

```bash
# Tải thư viện và build project
cmd /c "gradlew.bat assembleDebug 2>&1"
```
